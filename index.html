Main

- Master Switch: turns the whole Script ON/OFF

- FOV: modifies your camera FOV (only while batting)
  > FOV Value: value you want your FOV to be

- SPEED: modifies your LocalPlayer walk speed
  > Speed Boost: distance in studs added every physics simulation

Aimbot

- Aimbot: turns aimbot ON/OFF

- Mode (Mouse, Key): mode for input method of aimbot
  > "Mouse": uses the Mouse RightClick as input
  > "Key": uses the Keybind below as input

- Key: keybind for the mode "Key" above

- Smoothness: smoothing of mouse movement (higher = faster)

- Offsets: offsets the aiming target
  > Offset X: (higher = right, lower = left)
  > Offset Y: (higher = down, lower = up)

- Ball Check: detects if ball is in strikezone (doesn't predict !)

- Auto Hit: hits ball at the right timing (start aimlocking when ball esp appears)
  > Auto Hit Delay: dynamic delay calculated using time and speed of ball (higher = sooner, lower = later)

Visuals (2D Drawings)

- Ball ESP: draws a 2D circle around the ball
  > Color: color of the circle

- Strikezone ESP: draws a 2D rectangle around the strikezone
  > Color: color of the rectangle

Settings

PS: "End" key to toggle UI


- UI Position: position of the menu on screen

- Save Settings: saves current option states

- Destroy: exists HCBB X safely
